### Course Philosophy: Concept Extension  
Rather than treating econometrics and machine learning as separate disciplines, this course follows a **Concept Extension** approach:

- Start with foundational statistical tools (e.g., linear regression, hypothesis testing)
- Extend them using modern machine learning techniques (e.g., Lasso, regularization, cross-validation)
- Emphasize both **causal inference** and **predictive performance**

Through these projects, I am learning how to:
- Balance interpretability with prediction accuracy  
- Evaluate models using economic intuition and empirical rigor  
- Apply data-driven tools to real-world economic questions
